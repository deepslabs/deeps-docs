---
title: Wallet Setup
description: How to set up a wallet to interact with SafeX
---

To interact with SafeX and manage Token on this network, setting up a MetaMask wallet is essential. Follow these steps to configure MetaMask for SafeX:

### Installing Wallet

Install the wallet browser extension or mobile app if you haven't already.

* MetaMask: [https://metamask.io/download](https://metamask.io/download/)
* OKX: [https://www.okx.com/download](https://www.okx.com/download/)
* Bitget: [https://web3.bitget.com/en/wallet-download](https://web3.bitget.com/en/wallet-download/)
* TP: [https://www.tokenpocket.pro/en/download/app](https://www.tokenpocket.pro/en/download/app)

### Wallet Import or Creation

Import existing EVM wallets into the wallet or create new wallets specifically for managing Token on SafeX.

### Network Configuration

Configure the network to interact with the SafeX. Add the network [details](/docs/develop-guide/network-configuration), including the RPC endpoints and chain ID, to enable seamless communication.
